In questa sezione connetterai il tuo repository GitHub con Netlify. Netlify userà il tuo progetto per costruire e pubblicare online il tuo sito sul web ogni volta che farai un nuovo commit al tuo codice.

:::tip[useremo...]
Questo tutorial usa **Netlify** ma puoi optare per un servizio di hosting a tua scelta per pubblicare il tuo sito su internet. 
:::

## Crea un nuovo sito Netlify

1. Crea un account gratuito su [Netlify](https://netlify.com) se ancora non ne hai uno. 

    Prendi nota del tuo username. Potrai vedere il tuo pannello di controllo a tutti i siti che hai creato all'indirizzo `https://app.netlify.com/teams/username`

2. Fai clic su <kbd>Add new site</kbd> > <kbd>Import an existing project</kbd>. 

    Ti verrà richiesto di connetterti a un servizio Git. Scegli GitHub e segui i passi che ti vengono mostrati a video per autenticare il tuo account GitHub. Poi scegli il repository GitHub del tuo progetto Astro dall'elenco mostrato.
    Poi scegli il repository GitHub del tuo progetto Astro dall'elenco mostrato.

3. Come passo finale, Netlify ti mostrerà le impostazioni del sito della tua app. I valori predefiniti dovrebbero andare bene per il tuo progetto Astro, quindi scorri in basso e fai clic su <kbd>Deploy site</kbd>.

Complimenti, ora hai un sito Astro!

## Cambia il nome del tuo progetto

Nell'anteprima del tuo sito su Netlify noterai che il tuo progetto avrà un nome generato casualmente, e l'URL del tuo sito sarà nella forma `https://nome-progetto-123456.netlify.app`. Se vuoi puoi cambiare il nome in qualcosa di più semplice da memorizzare, e l'URL si aggiornerà automaticamente di conseguenza.

## Visita il tuo nuovo sito

Fai clic sull'URL nella pagina di impostazioni del tuo sito, o digitalo direttamente nel browser per vederlo online.
